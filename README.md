3/26 (목)

먼저 Mnist와 CIFAR-10 데이터를 학습하는 두 모델, MnistCNN과 CIFARCNN으로 나누어 코드를 작성하였습니다.
이 때 만약 사전 학습된 모델의 가중치가 없다면 코드는 학습을 진행하고,
사전 학습된 모델의 가중치가 있다면 코드는 학습을 진행하지 않고 평가만 진행하여 불필요한 학습 과정을 제거합니다.
-> 이 부분의 경우 로컬 환경에서 실험을 돌릴 때 편의를 위해 작성하였습니다.

초기 모델의 경우 MNIST는 99.36%의 정확도를 달성하였지만 CIFAR-10의 경우 약 75%의 정확도를 달성하여 CIFAR-10 모델을 대대적으로 수정하여 추가 실험을 진행하였습니다. 
1. 기존 2층으로 구성되어 있던 convolution layer를 3층으로 확장하였습니다. 이와 더불어 채널수도 256 차원으로 늘려 더 복잡한 특징을 추출할 수 있게 하였습니다.
2. Dropout 비율을 조절하여 각 블록마다 점진적으로 증가하게 만들어 과적합을 방지하였습니다.
3. Classifier를 256 -> 512로 늘려서 분류 능력을 향상 시키고자 하였습니다.
4. Optimizer를 Adam에서 SGD로 변경하였습니다.
5. lr = 0.001 -> 0.1 로 수정하였습니다.

결과적으로 MNIST 모델의 경우 99.36%, CIFAR-10 모델의 경우 84.56%의 test acc를 달성하였습니다.
