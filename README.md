# trustworthy-ai-assignment1

Readme 정보와 더불어 개발 과정을 작성하였습니다.

실행 환경 설정

pip3 install -r requirements.txt

실행 방법

python test.py

test.py를 실행하면 자동으로 다음 작업이 순서대로 진행됩니다.
1. MNIST 모델 학습 또는 불러오기 (목표 정확도 >= 95%)
2. CIFAR-10 모델 학습 또는 불러오기 (목표 정확도 >= 80%)
3. eps = 0.05, 0.1, 0.2, 0.3 에 대해 4가지 공격 방법 실행
4. 공격 성공률 출력 및 시각화 결과를 results/ 폴더에 저장 (단, eps 별로 따로 저장이 되지는 않습니다. eps 값에 따라 따로 폴더를 나눈 것은 결과를 확인한 뒤 가독성을 위해서 직접 분리하였습니다.)

사전 학습된 모델 파일 (mnist_model.pth, cifar_model.pth)이 이미 존재하면 학습을 건너뛰고 저장된 모델을 불러옵니다.
실험 결과 MNIST의 경우 99.25%, CIFAR-10의 경우 80.08% 달성하였습니다.

모델 구조

MNIST -> 2층 CNN
CIFAR-10 -> 3블록 CNN (BatchNorm + Dropout)

공격 방법

FGSM Targeted : 타겟 공격, gradient를 이용해 특정 클래스로 오분류 유도 (1회 스텝)
FGSM Untargeted : 비타겟 공격, gradient를 이용해 정답 클래스에서 멀어지도록 유도 (1회 스텝)
PGD Targeted : 타겟 공격, eps-ball 투영을 적용한 반복적 FGSM으로 특정 클래스 유도
PGD Untargeted : 비타겟 공격, eps-ball 투영을 적용한 반복적 FGSM으로 정답 클래스에서 멀어지도록 유도
Targeted : 원래 라벨 + 1 로 유도

3/26 (목)

먼저 Mnist와 CIFAR-10 데이터를 학습하는 두 모델, MnistCNN과 CIFARCNN으로 나누어 코드를 작성하였습니다.
이 때 만약 사전 학습된 모델의 가중치가 없다면 코드는 학습을 진행하고,
사전 학습된 모델의 가중치가 있다면 코드는 학습을 진행하지 않고 평가만 진행하여 불필요한 학습 과정을 제거합니다.
-> 이 부분의 경우 로컬 환경에서 실험을 돌릴 때 편의를 위해 작성하였습니다.

초기 모델의 경우 MNIST는 99.36%의 정확도를 달성하였지만 CIFAR-10의 경우 약 75%의 정확도를 달성하여 CIFAR-10 모델을 대대적으로 수정하여 추가 실험을 진행하였습니다. 
1. 기존 2층으로 구성되어 있던 convolution layer를 3층으로 확장하였습니다. 이와 더불어 채널수도 256 차원으로 늘려 더 복잡한 특징을 추출할 수 있게 하였습니다.
2. Dropout 비율을 조절하여 각 블록마다 점진적으로 증가하게 만들어 과적합을 방지하였습니다.
3. Classifier를 256 -> 512로 늘려서 분류 능력을 향상 시키고자 하였습니다.
4. Optimizer를 Adam에서 SGD로 변경하였습니다.
5. lr = 0.001 -> 0.1 로 수정하였습니다.

결과적으로 MNIST 모델의 경우 99.36%, CIFAR-10 모델의 경우 84.56%의 test acc를 달성하였습니다.
